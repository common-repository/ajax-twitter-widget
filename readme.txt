A widget that allows to update with the last tweets the sidebar of your Wordpress site. 

== Description ==

The Ajax Twitter widget allows to dynamically and asyncronously display the last tweets. The administrator can even set such preset styles like the font family, the font size, the text color, the kind of image displayed on top and the border color. This widget can be easily installed and displayed in any right or left sidebar. Translated in Portugese, Spanish, English, Italian and Franch.
